# Contributing

## Setup

1. Use Node.js 20+.
2. Install dependencies: `npm install`.
3. Copy `.env.example` to `.env` and provide required values.

## Development

1. Run tests before opening a PR: `npm test`.
2. Keep changes scoped and include clear commit messages.
3. Update `README.md` when behavior or configuration changes.

## Pull Requests

1. Describe what changed and why.
2. Include test evidence or explain why tests were not needed.
3. Keep secrets out of code, logs, and screenshots.
