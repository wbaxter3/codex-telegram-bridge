# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected Behavior

1. Be considerate in language and tone.
2. Assume good intent and discuss ideas on technical merit.
3. Accept feedback gracefully and focus on collaboration.

## Unacceptable Behavior

1. Harassment, discrimination, or personal attacks.
2. Doxxing or sharing private information without consent.
3. Disruptive behavior that prevents productive collaboration.

## Enforcement

Project maintainers may remove comments, reject contributions, or block access for conduct violations.
